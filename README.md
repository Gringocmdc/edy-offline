# EDY Offline v1.1 — Fichas y Checklists

## Novedades
- Checklists operativas y personalizadas.
- Checklist de evacuación integrada al Modo Crisis.
- Diagnóstico local de preparación con recomendaciones.
- Puntaje integrado: operaciones, inventario, ubicaciones, mantenimiento y listas.
- Fichas profesionales de inventario con marca/modelo, serie y frecuencia de revisión.
- Fotografía local comprimida para cada elemento mediante IndexedDB.
- Botón “Revisado hoy” que calcula la próxima revisión.
- Respaldo v1.1 con checklists y fotografías.
- Barra inferior de navegación para uso tipo aplicación.

## Privacidad reforzada
El archivo público `inventory.json` queda vacío. Los datos personales, fotografías, números de serie y ubicaciones se guardan únicamente en el dispositivo. La actualización conserva el inventario local existente del mismo navegador.
