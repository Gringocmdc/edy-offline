# EDY 1.0 Beta — Sistema Integrado

## Novedades principales
- Modo Crisis con interfaz simplificada.
- Historial local de actividad.
- Mantenimiento de equipos y revisiones.
- Fecha de compra, garantía, responsable y última revisión.
- Respaldo completo exportable a JSON.
- Importación y restauración de respaldos.
- Resumen diario en la pantalla principal.
- Integración entre inventario, mapa, operaciones, asistente y protocolos.
- Conserva los datos locales de versiones anteriores.

## Seguridad
Los respaldos pueden contener datos privados. Guardarlos en un lugar seguro.
